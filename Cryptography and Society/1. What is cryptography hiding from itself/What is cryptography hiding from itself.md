# What is cryptography hiding from itself?

## Abstract

從科學哲學與政治哲學的角度分析歐盟 Chat Control 提案，指出強制客戶端掃描不僅是技術問題，更是一種知識論上的權力重構——將「安全」從道德判斷轉化為偵測閾值，將政治決定偽裝為參數調整，並在使用者裝置上建立常態化的監控基礎設施。

## Summary

### 密碼學重新分配權力

演講以 Phil Rogaway 的論文《密碼學工作的道德特質》為起點，引用其核心觀點：**密碼學重新分配權力（Cryptography rearranges power）**。即使在進行純形式化工作時，我們創造的成品（artifacts）也會改變能力、激勵結構、以及誰對誰擁有影響力。當一項政策提案試圖重塑加密通訊時，密碼學社群的回應不應僅限於「這破壞了端對端加密」或「這不可行」，還應追問：**提出了什麼樣的權力安排？什麼樣的證據標準被偽裝成了技術細節？**

### 端對端加密的社會意義

形式上，端對端加密意味著服務提供者無法讀取傳輸中的密文。但社會上，人們理解的含義更廣：**這個通道不是被設計來例行性地檢查我的私人生活**。Chat Control 壓力測試了這個社會意義——如果在保持線上加密的同時強制偵測，檢查只是被重新定位到端點（加密前或解密後），使裝置成為執法管道的一部分。

### Chat Control 的架構邏輯

Chat Control 是歐盟的一項提案，推動服務商建立「偵測—通報—移除」的管線來處理兒童性虐待內容（CSAM）。其關鍵特徵是**技術中立姿態**——不豁免端對端加密服務，義務以結果表達（實現偵測），由服務商自行決定如何實現，即使無法存取明文。

**架構上無法迴避的一點**：如果要求內容檢查，且維持端對端加密（提供者看不到明文），那檢查必須移到明文存在的地方——即端點。這就是客戶端掃描成為合規候選方案的原因。

### 偵測閾值 τ 的政治本質

**簽章式偵測**（已知內容的雜湊比對）與**分類器式偵測**（未知內容的分類）是完全不同的世界。分類器需要**閾值 τ**——τ 不是純技術常數，而是社會劃分「什麼證據足以證明介入正當性」的那條線。

演講者透過現場投票展示：選擇較低的 τ（抓更多，接受更多誤判）還是較高的 τ（保護無辜，接受更多漏檢）？觀眾的選擇本身就是**治理行為**——政治決定被包裝成參數調整。

**核心論點**：
- 誤判的代價不對稱——假陽性（無辜者被懷疑、通報、承受後果）與假陰性（漏檢）是不同性質的傷害
- 選擇 τ 就是在**分配傷害**
- 安全被轉化為認識論建構（epistemic construct）：原本的政治問題「什麼傷害不可容忍？」被轉化為技術問題「什麼信號超過偵測閾值？」

### 科學哲學框架

**Sheila Jasanoff 的「傲慢技術」（technologies of hubris）**：治理透過將道德風險轉化為可計算的控制指標來穩定確定性。模型是不確定的，世界是混亂的，對手會適應——但監管需要觸發器和執法，所以將不確定預測轉化為看似確定的東西（旗標、通報、工作流程）。度量成為合法性來源：「我們測量了，所以我們理性地行動了。」

**以賽亞·柏林的消極/積極自由**：Chat Control 以積極自由的修辭（保護兒童、保護社會）為框架，但以干預個人通訊的方式實現——消耗消極自由作為手段。柏林的警告：「為了你好」的邏輯可能成為強制的理據。

**Langdon Winner 的「技術物有政治性」**：技術系統體現權力與權威的安排。客戶端掃描是典型案例——透過在個人裝置中強制掃描演算法，將政治選擇銘刻於技術中。一旦此能力存在，就可擴展到其他目標（今天 CSAM，明天可能是異見者）。

### 未完成的科學（Undone Science）

Chat Control 領域中被系統性忽略的問題：
- 歐盟規模的假陽性數量是多少？無辜者的救濟機制如何？
- 對手適應掃描器後的均衡狀態是什麼？
- 在數十億裝置上強制部署掃描邏輯加更新機制，創造了什麼新攻擊面？
- 誰控制模型更新？資料庫成員資格？審計？申訴？

系統被結構化為：**缺失的答案被視為可接受的附帶損害**——這本身是一個道德選擇。

### 密碼學社群的角色

密碼學家特別擅長預測二階失敗模式。如果能清晰地闡述這些模式，就能幫助政策避免建造脆弱的基礎設施。演講以論文的結語作結：**「當恐懼治理時，自由收縮。我們的任務是建造不以服從為代價的保護基礎設施。」**

## 與 TWCA 臺灣網路認證的關聯

1. **客戶端掃描對 PKI 信任模型的衝擊**：如果端點裝置被要求執行第三方檢查程式碼，這改變了 TWCA 的 PKI 信任邊界。TWCA 簽發的憑證所保護的端對端加密通道，若端點本身成為監控基礎設施的一部分，則憑證所提供的安全保證在社會意義上被削弱。

2. **監管合規與隱私權的平衡**：TWCA 作為受政府監管的認證中心，未來可能面臨類似的監管壓力——要求在加密基礎設施中嵌入偵測或審查機制。本研究提供的哲學框架（區分技術可行性、認識論正當性、倫理合理性）有助於 TWCA 在面對此類壓力時進行系統性分析。

3. **偵測閾值與數位身分驗證的類比**：TWCA 的身分驗證服務同樣涉及「什麼證據足以證明身分」的閾值決策。本研究對 τ 的政治本質分析，提醒 TWCA 在設計驗證流程時，閾值的設定不僅是技術問題，也是一個攸關使用者權益的治理決策。
